pobi의 설계 방식대로 TDD를 구현하다보니 TDD를 좀 더 완성도 높은 방식으로 체험해 볼 수는 있었지만 나만의 설계 방식이 유연하게 적용되는데까지 시간이 많이 걸렸다. 항상 사고가 갇히는 것을 경계하고, 이 때엔 내 생각대로 처음 부터 다시 설계해보는 것이 좋은 방법이 되는 것 같다
